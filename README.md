# LisaDesktop GitHub Documentation

Welcome to the main repository for LisaDesktop. This project focuses on building an operating system environment featuring the custom "lumpy" package manager based on alias apt. The goal is to provide a wide variety of desktop choices while testing non-immutable system designs and lightweight alternatives for older computers.

## Core Features
* **Lumpy Package Manager:** A custom management system built to handle applications via alias apt configurations.
* **on-Immutable Infrastructure:** The main edition uses an unchangeable root filesystem to prevent system breakage and ensure stability.
* **Desktop Diversity:** Multiple official desktop editions are available to suit specific performance preferences and user choices.
* **Flatpak:** Flatpak is our choice users love flatpak becuase its fast if users want to install their apps and own choices.

## Available Editions Overview
The project is split into different versions depending on your hardware limits and preferred user interface layout:
1. **LisaDesktop Main Edition:** Features the standards system structure using the custom package manager setup.
3. **LisaDesktop MATE / Xfce Editions:** Lightweight options designed specifically to provide speed on older computers.
4. **LisaDesktop Cinnamon / Fluxbox Editions:** Visual options that balance modern interface elements with low hardware requirements.
5. **Lisa Desktop Regolith Edition:** A modern tiling window manager layout for efficient keyboard-driven workflows.
6. **Lisa Desktop AwesomeWM Edition:** A modern tiling window manager layout for efficient keyboard-driven workflows.
